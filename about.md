## About This Catalog

This site catalogs **Broadcom mainframe solutions** that integrate with one or both of:

- **Zowe API Mediation Layer (API ML)** — the open-source gateway, discovery service, and API catalog for z/OS REST services.
- **VS Code** — extensions published under the [`broadcomMFD`](https://marketplace.visualstudio.com/publishers/broadcomMFD) publisher or the [`Zowe`](https://marketplace.visualstudio.com/publishers/Zowe) publisher.

## Zowe Conformance Program

The [Zowe Conformance Program](https://www.openmainframeproject.org/our-projects/zowe-conformance-program/)
certifies ISV products in the following categories:

- **Zowe Conformant — API Mediation Layer** — product integrates correctly with API ML
- **Zowe Conformant — App Framework** — product provides a Zowe Desktop plug-in
- **Zowe Conformant — CLI** — product provides a conformant Zowe CLI plugin
- **Zowe Conformant — Explorer for VS Code** — product provides a conformant Zowe Explorer extension or integration

Solutions in this catalog that have earned Zowe Conformant status show a **Conformant** badge on their detail page.

## Zowe Background

[Zowe](https://zowe.org) is an open-source project hosted by the
[Open Mainframe Project](https://openmainframeproject.org/) (Linux Foundation).
Broadcom is a founding sponsor and primary contributor. Zowe provides:

- API Mediation Layer (server-side gateway)
- Zowe Application Framework (web desktop)
- Zowe CLI (command-line interface)
- Zowe Explorer (VS Code extension)
- Client SDKs (Node.js, Java, Kotlin, Python)
